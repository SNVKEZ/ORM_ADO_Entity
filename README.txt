В данной программе реализована орм на основе фреймворка Entity. БД создана на логике владелец - автомобиль. UI - веб на ASP.NET. В бд с помощью языка SQL непосредственно 
запросом добавляем все данные таблиц. На самой веб-форме уже можем удалить, изменить, добавить элемент. Совместная работа таблиц бд показана колонкой "Машины" 
которая выводит инфу благодаря внешнему ключу бд.
